Protectli V1410 Home Lab

Personal cybersecurity lab using Proxmox + pfSense + Wieguard to simulate enterprise network security architecture.

Project started: Feb 2026
1. Install Rufus if you do not already have it. (https://rufus.ie/en/)
1. Flash Proxmox ISO onto a USB installable media device (https://www.proxmox.com/en/downloads/proxmox-virtual-environment)
2. Install NVME M.2 drive into V1410 (recommended over On-board 32GB eMMC)
3. Plug the USB into V1410 and boot with Ethernet plugged in from the modem
4. Follow the installation steps
